📝 Инструкция

Как настроить:

  1. Откройте Google Таблицу
  2. Расширения → Apps Script
  3. Вставьте код из поля выше
  4. Развернуть → Новое развертывание
  5. Тип: Веб-приложение, Доступ: Все
  6. Скопируйте URL и вставьте выше
  7. Нажмите "Тест" для проверки
